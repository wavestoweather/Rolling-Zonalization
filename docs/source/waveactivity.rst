Local Wave Activity
===================

Finite-amplitude local wave activity as a Rossby wave amplitude diagnostic (`Huang and Nakamura 2016 <https://doi.org/10.1175/JAS-D-15-0194.1>`_; `Ghinassi et al. 2018 <https://doi.org/10.1175/MWR-D-18-0068.1>`_).
The implementation provided here fills a few gaps for primitive equations LWA on isentropes left by the (otherwise) much more comprehensive `falwa Python package <https://github.com/csyhuang/hn2016_falwa>`_ for the QG formalism by C. S. Y. Huang.
See also P. Ghinassi's `original implementation <https://github.com/paologhinassi/RWPtools>`_ of primitive equations LWA.
